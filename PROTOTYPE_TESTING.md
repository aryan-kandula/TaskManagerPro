# Prototype Testing Comparison

## Version 1: Claude (Primary Tool)
- ✅ Works:
  - Generated structured and readable HTML/CSS/JS code
  - Implemented task management functionality correctly
  - Produced clean UI layout with logical sections
  - Included comments and understandable structure
- ❌ Broken:
  - Some advanced features required refinement (timer logic, AI assistant behavior)
  - Minor UI inconsistencies needed manual adjustments
- 💭 Observations:
  - Claude produced well-organized and maintainable code
  - Easier to iterate with due to clear explanations
  - Good balance between functionality and readability

---

## Version 2: ChatGPT (Secondary Comparison)
- ✅ Works:
  - Generated more visually polished UI components
  - Provided faster feature additions
- ❌ Broken:
  - Some generated code required more debugging
  - Less structured compared to Claude output
- 💭 Observations:
  - Strong for UI/UX ideas
  - Required more iteration for stable functionality

---

## Decision

I chose Claude as my primary development tool because:
- It generated more structured and readable code
- Easier to debug and iterate on
- Better aligned with building a maintainable single-page application

ChatGPT was useful for refining UI ideas and improving specific features, but Claude provided a stronger foundation.
