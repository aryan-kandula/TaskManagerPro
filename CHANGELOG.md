# Changelog

All notable changes to Task Manager Pro will be documented in this file.

## [2.0.0] - 2026-01-31

### Added
- Enhanced AI assistant with natural language date parsing
- Support for date range queries ("tasks until Feb 10")
- Support for specific date queries ("tasks on March 15")
- This week/next week query support
- Duplicate task handling - can now add the same task multiple times
- Toggle button for past due tasks section
- Improved task card layout with better text handling
- Consistent checkbox alignment regardless of text length
- Word wrapping for long task descriptions and notes
- Separate upcoming and past due task sections

### Changed
- Reorganized dashboard to show upcoming tasks first
- Past due tasks now hidden by default with toggle button
- Improved task card flexbox layout for better responsiveness
- Enhanced text overflow handling in task cards
- Updated AI assistant welcome message

### Fixed
- Task checkbox alignment issues with long descriptions
- Text overflow in task cards
- Layout breaking with very long task notes
- Inconsistent spacing in task cards

## [1.0.0] - 2026-01-01

### Added
- Initial release
- Task creation with description, date, priority, and notes
- Dashboard with statistics
- Task completion tracking
- Google Calendar integration
- Canvas LMS integration
- ICS file import
- AI assistant with basic queries
- Export to JSON and CSV
- Dark glassmorphic UI design
- Local storage persistence
- Priority system (High/Medium/Low)
- Task deletion
- Responsive design
